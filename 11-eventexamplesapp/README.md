# Exercise 11 - eventexamplesapp

## Objective
Create a react app "eventexamplesapp" demonstrating event handling:
- `Increment` button increments the counter AND shows a static "Hello"
  alert (invokes multiple methods).
- `Decrement` button decrements the counter.
- `Say Welcome` button invokes a function passing "welcome" as an
  argument.
- `OnPress` button handles a synthetic click event and alerts
  "I was clicked".
- `CurrencyConvertor` component converts INR to EUR on submit, using
  `handleSubmit`.

## How to run
```
npm install
npm start
```
