# Exercise 5 - Cohort Dashboard (CSS Modules)

## Objective
Style the `CohortDetails` component using a CSS Module
(`CohortDetails.module.css`):
- `.box` class: width 300px, `display: inline-block`, 10px margin,
  10px top/bottom padding, 20px left/right padding, 1px black border,
  10px border radius.
- `dt` tag selector uses `font-weight: 500`.
- The `<h3>` cohort code is green when the cohort status is "ongoing"
  and blue otherwise.

## How to run
```
npm install
npm start
```
