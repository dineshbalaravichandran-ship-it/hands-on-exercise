# React JS Hands-On Solutions

Solutions to the 19 ReactJS Hands-On Lab exercises from the
`Digital-Nurture-JavaFSE` course (Deepskilling / React module).

Each numbered folder is a standalone, runnable React app (created with
Create React App-style structure) containing its own `README.md` that
explains the exercise objective and how to run it.

| # | Folder | Topic |
|---|--------|-------|
| 1 | `01-myfirstreact` | SPA & React setup, create-react-app |
| 2 | `02-StudentApp` | Class components |
| 3 | `03-scorecalculatorapp` | Function components & styling |
| 4 | `04-blogapp` | Component lifecycle (`componentDidMount`, `componentDidCatch`) |
| 5 | `05-cohort-dashboard` | CSS Modules |
| 6 | `06-TrainersApp` | React Router (`BrowserRouter`, `Routes`, `useParams`) |
| 7 | `07-shoppingapp` | Class components & arrays |
| 8 | `08-counterapp` | React state |
| 9 | `09-cricketapp` | ES6 features (`map`, `filter`, destructuring, spread) |
| 10 | `10-officespacerentalapp` | JSX & inline CSS |
| 11 | `11-eventexamplesapp` | Event handling & synthetic events |
| 12 | `12-ticketbookingapp` | Conditional rendering |
| 13 | `13-bloggerapp` | Multiple ways of conditional rendering |
| 14 | `14-employee-theme-context` | Context API (`createContext`, `useContext`) |
| 15 | `15-ticketraisingapp` | React forms |
| 16 | `16-mailregisterapp` | Form validation |
| 17 | `17-fetchuserapp` | Consuming REST APIs |
| 18 | `18-cohort-unit-testing` | Unit testing with Jest & Enzyme |
| 19 | `19-gitclientapp` | Isolated unit testing with Jest mocks (axios) |

## Running any exercise

```bash
cd 0X-folder-name
npm install
npm start
```

For the testing exercises (18 and 19):

```bash
cd 18-cohort-unit-testing   # or 19-gitclientapp
npm install
npm test
```
