# Exercise 10 - officespacerentalapp (JSX)

## Objective
Create a react app "officespacerentalapp" using JSX to:
- Create an element for the page heading.
- Create an attribute-based `<img>` element for the office image.
- Create an `office` object and display its Name, Rent and Address.
- Create a list of office objects and loop through them with `map()`.
- Apply inline CSS so the rent shows red when below 60000 and green
  when above 60000.

## How to run
```
npm install
npm start
```
