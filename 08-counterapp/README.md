# Exercise 8 - counterapp

## Objective
Create a react app "counterapp" with component `CountPeople` that uses
the constructor and `state` to store `entrycount` and `exitcount`.
- `Login` button calls `updateEntry()` which increments `entrycount`.
- `Exit` button calls `updateExit()` which increments `exitcount`.

## How to run
```
npm install
npm start
```
