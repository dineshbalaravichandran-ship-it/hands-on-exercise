# Exercise 15 - ticketraisingapp (React Forms)

## Objective
Create a react app "ticketraisingapp" with a `ComplaintRegister`
component that has a controlled form: a textbox for employee name and
a textarea for the complaint. `handleSubmit` submits the complaint,
generates a random reference number, and shows it in an alert box.

## How to run
```
npm install
npm start
```
