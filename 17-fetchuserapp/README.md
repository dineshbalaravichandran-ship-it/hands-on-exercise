# Exercise 17 - fetchuserapp (Consuming REST APIs)

## Objective
Create a react app "fetchuserapp" with a `Getuser` component that
fetches a random user from `https://randomuser.me/api/` inside
`componentDidMount()` and displays the title, first name and picture
of the user.

Note: the exercise text references `https://api.randomuser.me/`; the
correct/working endpoint for the Random User API is
`https://randomuser.me/api/`, which is used here.

## How to run
```
npm install
npm start
```
