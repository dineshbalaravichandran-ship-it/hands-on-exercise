# Exercise 16 - mailregisterapp (Form Validation)

## Objective
Create a react app "mailregisterapp" with a `register.js` component
containing a form for name, email and password, validated on both
`onChange` and `onSubmit`:
1. Name must have at least 5 characters.
2. Email must contain `@` and `.`.
3. Password must have at least 8 characters.

Validation error messages are shown inline under each field.

## How to run
```
npm install
npm start
```
