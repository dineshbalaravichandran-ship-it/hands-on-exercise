# Exercise 9 - cricketapp (ES6 Features)

## Objective
Create a react app "cricketapp" that demonstrates ES6 features:
- `ListofPlayers`: an array of 11 players rendered with `map()`, and
  filtered with `filter()` (arrow function) to show players scoring
  below 70.
- `IndianPlayers`: shows odd/even team players using array
  destructuring, and merges `T20players` and `RanjiTrophyPlayers`
  arrays using the ES6 spread operator.
- `App.js` toggles between the two components using a `flag` variable.

## How to run
```
npm install
npm start
```
