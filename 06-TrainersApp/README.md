# Exercise 6 - TrainersApp (React Router)

## Objective
Build a SPA with `react-router-dom` that lists trainers and lets you
view details of each one:
- `/` renders the `Home` component.
- `/trainers` renders `TrainersList`, showing trainer names as links.
- `/trainers/:id` renders `TrainerDetails`, reading `id` via
  `useParams()` and looking up the trainer in the mock data.

## How to run
```
npm install
npm start
```
