# Exercise 4 - blogapp

## Objective
Create a react app "blogapp" demonstrating component lifecycle hooks.
- `Post` is a plain JS class describing a post.
- `Posts` is a class component that fetches posts from
  `https://jsonplaceholder.typicode.com/posts` inside `loadPosts()`.
- `componentDidMount()` calls `loadPosts()`.
- `render()` displays the title and body of each post.
- `componentDidCatch()` displays any rendering error as an alert.

## How to run
```
npm install
npm start
```
