# Exercise 13 - bloggerapp (Multiple Ways of Conditional Rendering)

## Objective
Create a react app "bloggerapp" with three components (`BookDetails`,
`BlogDetails`, `CourseDetails`), each demonstrating a different way of
doing conditional rendering in React:
1. `if / else` statement inside the component (`BookDetails`).
2. Logical `&&` operator (`BlogDetails`).
3. Ternary `? :` operator (`CourseDetails`).
4. `switch` statement wrapped in an IIFE (`App.js`).

## How to run
```
npm install
npm start
```
