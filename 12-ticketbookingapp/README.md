# Exercise 12 - ticketbookingapp (Conditional Rendering)

## Objective
Create a react app "ticketbookingapp" where guest users see flight
listings (`GuestPage`) and logged-in users see the booking page
(`UserPage`). `Login`/`Logout` buttons toggle between the two views
using conditional rendering based on state.

## How to run
```
npm install
npm start
```
